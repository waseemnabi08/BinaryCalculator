Binary Calculator

Welcome to the Binary Calculator documentation. This page provides details on how to use the calculator and its features.

Binary Operations

This section covers basic binary arithmetic operations.

Addition

Route: /

Method: POST

Description: Perform binary addition of two binary numbers.

Parameters:

  • num1: The first binary number (string).
  • num2: The second binary number (string).

Example: To add binary numbers, enter num1 and num2, select "Addition" as the operation, and submit the form.

Subtraction

Route: /

Method: POST

Description: Perform binary subtraction of two binary numbers.

Parameters:

  • num1: The first binary number (string).
  • num2: The second binary number (string).

Example: To subtract binary numbers, enter num1 and num2, select "Subtraction" as the operation, and submit the form.

Conversion Operations

This section explains how to perform various binary conversions.

Binary to Decimal Conversion

Route: /convert

Method: POST

Description: Convert a binary number to its decimal equivalent.

Parameters (POST):

  • input: The binary number to convert (string).
  • conversionType: Select "Binary to Decimal" as the conversion type.

Example: Enter a binary number, select "Binary to Decimal" as the conversion type, and submit the form to perform the conversion.

Decimal to Binary Conversion

Route: /convert

Method: POST

Description: Convert a decimal number to its binary equivalent.

Parameters (POST):

  • input: The decimal number to convert (integer).
  • conversionType: Select "Decimal to Binary" as the conversion type.

Example: Enter a decimal number, select "Decimal to Binary" as the conversion type, and submit the form to perform the conversion.

Floating-Point Conversion

Route: /convert

Method: POST

Description: Convert between floating-point representations (binary to float and vice versa).

Parameters (POST):

  • input: The floating-point number to convert (string).
  • conversionType: Select the desired conversion type (e.g., "Binary to Float").

Example: Enter a floating-point number, select the conversion type, and submit the form to perform the conversion.

Two's Complement

Route: /convert

Method: POST

Description: Compute the two's complement of a binary number.

Parameters (POST):

  • input: The binary number to complement (string).
  • conversionType: Select "Two's Complement" as the conversion type.

Example: Enter a binary number, select "Two's Complement" as the conversion type, and submit the form to compute the result.
